前言

本文适合已经有一门高级语言基础的同学观看，字不多，大部分把语法以代码的方式呈现。

print功能

  1. print(1) 输出：1
  2. print(“……”/‘……’) 输出：……
  3. print(‘I’m a boy) 报错，正解如下：
    • print(“I’m a boy”)用双引号保住句子
    • 用单引号包住句子，在‘号前面加上转义字符
  4. print(‘age’+’18’) 输出：age18
  5. print(‘age’+18) 报错，正解如下：
    • print(‘age’+’18’)
    • print(‘age’+str(18))
  6. print(1+1) 输出：2
  7. print(‘1’+1) 报错，正解如下：
    • print(int(‘1’)+1) 输出：2
  8. print(int(‘1.1’)+1) 报错，int()只能转化整数，不能转化小数。正解如下：
    • print(float(‘1.1’)+1) 输出：2.1
  9. print(1,2,3) 自动带空格分隔，输出1 2 3
    • print(‘a:’,1,’b’,2) 输出：a: 1 b: 2
  10. print自动带换行回车
    • 取消换行print(……，end=””)即可。end=”……”……即输出后跟着的东西。print(1,end=”“)输出”1\“

运算符号

  1. python中的加(+)、减(-)、乘(*)、取余(%)和c/c++中符号意义相同，而python中的除(/)不是整除，整除为”//“
    • print(5/3) 输出：1.6666666666666667
  2. 平方符号。2的平方即为2**2
    • print(3**2) 输出3的平方：9
    • print(2**4) 输出2的四次方：16
  3. python不支持自增，即i++。支持i+=1。

变量

python中不需要int a = ……;直接赋值即可

  • a = 1, str = “abc”等
  • 同时赋值多个变量：a,b,c=1,2,3(print(a,b,c) = “1 2 3”)

正则表达式

正则表达式(Regular Expression，又称RegEx)，是用来匹配字符串的一种工具。爬取网页时常用。

1、简单的匹配

使用正则表达式则需要调用一个python的内置模块re。

如果 re.search() 找到了结果, 它会返回一个 match 的 object。 如果没有匹配到, 它会返回 None。

2、高级的匹配

匹配多种可能，使用[]

需要找到多个可能性的字符串，用[]将可能的字符包裹起来。比如[ab]表示要找的字符可以是a或b，在”……“的前面加上r表示这是一个正则表达式，不加则是一个普通的字符串。

正则表达式查询表

不一一举例。

注意re.search()只会返回第一次匹配到的结果

多行匹配

如果一个字符串有很多行，我们想用^来匹配开头字符。

如果想匹配的字符是在第二行的开头，用r"^I"是匹配不到的。这时候要使用 另外一个参数, 让 re.search() 可以对每一行单独处理. 这个参数就是 flags=re.M或者 flags=re.MULTILINE。

可以使用()为匹配到的内容分组。通过分组，我们可以定位匹配到的内容。
